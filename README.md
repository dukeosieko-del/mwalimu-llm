# MWALIMU AI

AI-Powered Educational Document Generator for the Kenyan Education System

## Features

- Smart Exams: Generate curriculum-aligned exams with varying difficulty levels
- Schemes of Work: Create detailed, standards-compliant schemes of work
- Lesson Plans: Develop comprehensive lesson plans with objectives and procedures
- Subject Notes: Generate concise, comprehensive subject notes
- Timetables: Design optimized timetables for schools
- Digital Diaries: Maintain comprehensive digital diaries for student tracking

## Getting Started

1. Visit the landing page to learn about MWALIMU AI
2. Click "Try MWALIMU AI" to access the chat interface
3. Select your preferred AI model (Grok, ChatGPT, DeepSeek, or Claude)
4. Start generating educational documents!

## Deployment

This project is configured for deployment on Hugging Face Spaces as a static website.
